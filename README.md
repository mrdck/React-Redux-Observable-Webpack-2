## Installing 
```
git clone https://github.com/mrdck/React-Redux-Observable-Webpack-2.git directory_name
cd directory_name
npm install
```

## Usage

To start server all what you need is to do :
```
npm start 
```
OR

```
npm run start
```

## Features

This starter pack contains :

* React
* Babel (with stage-0)
* React-redux
* Redux
* RxJS
* Redux-Observable
* Jest
